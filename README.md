# Estudos em Python e Fundamentos de Programação

Este repositório reúne exercícios e pequenos projetos que desenvolvi durante meus estudos em Python e lógica de programação.  
Os códigos aqui representam minha evolução prática, focada em resolver problemas usando programação.

Ao longo desses estudos, pratiquei principalmente:
- Variáveis e tipos de dados
- Operadores matemáticos e lógicos
- Estruturas de decisão (if / else)
- Estruturas de repetição (for e while)
- Listas, dicionários e manipulação de dados
- Criação de funções
- Organização e leitura de código

Os exercícios foram criados para transformar problemas em soluções programáveis, como cálculos, análises simples de dados e automações básicas.  
Este repositório faz parte da construção do meu portfólio como desenvolvedor em formação, com foco em evoluir para a área de dados e tecnologia.
